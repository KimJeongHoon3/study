클래스 초기화 블록

```java
class InitBlock {

static { /* 클래스 초기화 블럭 */ }

{ /* 인스턴스 초기화 블럭 */ }

// ...
}
```

- 클래스 초기화 블럭은 클래스가 메모리에 처음 로딩될 때 한번만 수행되며,
  - 클래스가 처음 로딩될 때 클래스 변수들이 자동적으로 메모리에 만들어지고, 곧바로 클래스 초기화 블럭이 클래스 변수들을 초기화하게 되는 것이다.
- 인스턴스 초기화 블럭은 생성자와 같이 인스턴스를 생성할 때 마다 수행된다.
